# Gemini Reader 功能优化方案

## 项目现状分析

### 已实现的核心功能 ✅
1. EPUB 电子书上传和阅读
2. 书籍元数据自动提取（标题、作者、封面）
3. 书库管理（网格展示、删除）
4. 字体设置（11种字体、10种大小）
5. 书签管理（添加、编辑、删除、跳转）
6. 阅读进度自动保存
7. 伪装成 Gemini AI 界面
8. 本地数据持久化

### 技术栈
- Vue 3 + Vite
- Pinia (状态管理)
- epub.js (EPUB 渲染)
- Tailwind CSS (样式)
- localForage (大文件存储)
- localStorage (元数据存储)

## 优化方案（按优先级排序）

## 🔥 高优先级功能（核心体验提升）

### 1. 阅读进度可视化
**问题**：用户不知道当前阅读到哪里了

**方案**：
- 在书籍卡片上显示阅读进度条
- 显示百分比（已读 45%）
- 在阅读器底部显示章节进度
- 在侧边栏书籍列表显示进度圆环

**实现难度**：⭐⭐
**用户价值**：⭐⭐⭐⭐⭐

```javascript
// 需要在 bookStore 中添加
{
  progress: 0.45, // 0-1 之间
  totalPages: 350,
  currentPage: 158
}
```

### 2. 目录导航（TOC）
**问题**：无法快速跳转到特定章节

**方案**：
- 点击侧边栏"搜索"图标打开目录面板
- 显示书籍章节树形结构
- 点击章节标题快速跳转
- 高亮当前所在章节

**实现难度**：⭐⭐⭐
**用户价值**：⭐⭐⭐⭐⭐

**代码提示**：已经有 `toc` 变量和 `showTOC` 状态，只需要创建 TOC 组件

### 3. 全文搜索
**问题**：无法在书中搜索关键词

**方案**：
- 点击"搜索"图标打开搜索面板
- 输入关键词实时搜索
- 显示搜索结果列表（上下文预览）
- 点击结果跳转到对应位置
- 高亮显示搜索词

**实现难度**：⭐⭐⭐⭐
**用户价值**：⭐⭐⭐⭐⭐

**代码提示**：已经有 `searchQuery`、`searchResults`、`isSearching` 变量

### 4. 阅读主题切换
**问题**：只有代码风格主题，长时间阅读眼睛疲劳

**方案**：
- 添加多种阅读主题：
  - 代码风格（当前）
  - 护眼模式（米黄色背景）
  - 夜间模式（深色）
  - 纸质书模式（白色背景）
  - 高对比度模式
- 在字体设置中添加主题选择
- 保存用户偏好

**实现难度**：⭐⭐⭐
**用户价值**：⭐⭐⭐⭐⭐

### 5. 笔记和高亮功能
**问题**：无法标记重要内容和添加笔记

**方案**：
- 选中文本后显示工具栏
- 支持多种颜色高亮
- 添加文本笔记
- 在书签面板中显示所有高亮和笔记
- 导出笔记功能

**实现难度**：⭐⭐⭐⭐⭐
**用户价值**：⭐⭐⭐⭐⭐
